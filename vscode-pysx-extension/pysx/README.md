# ReactXPy Language Support

ReactXPy Language Support adds syntax highlighting and editor features for **ReactXPy**, a Python-inspired JSX syntax for building reactive user interfaces.

## Features

- Syntax highlighting for `.pysx` files
- JSX-style tag support
- Python-like syntax recognition
- Bracket matching and auto-closing pairs

## About ReactXPy

ReactXPy combines:

- Python readability
- Component-based UI architecture
- React-style declarative interfaces
